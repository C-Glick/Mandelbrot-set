# Mandelbrot-set
A java program to generate a graph of the mandelbrot set

The base equation: z=z<sup>2</sup>+c

Learn more about the mandelbrot set here ---> [Wikipedia.org](https://en.wikipedia.org/wiki/Mandelbrot_set)
